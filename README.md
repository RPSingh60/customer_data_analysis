# Data Analytics Project

## Overview

This project demonstrates a complete end-to-end data analytics workflow. It starts with loading a raw dataset in Python, performing Exploratory Data Analysis (EDA), cleaning and preparing the data, analyzing it using SQL, building an interactive Power BI dashboard, and presenting the findings through a report and presentation.

The objective of this project is to transform raw data into meaningful insights that support data-driven decision-making.

---

## Dataset

The dataset contains structured business data used for analysis and visualization.

The analysis includes:

* Data loading
* Data cleaning
* Handling missing values
* Removing duplicates
* Data transformation
* Exploratory Data Analysis (EDA)
* SQL-based analysis
* Dashboard creation

* Imported the dataset into Python.
* Verified data types and dataset structure.

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter notebooks and run them in sequence:

   * Data Loading
   * EDA
   * Data Cleaning
4. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
5. Execute the SQL scripts.
6. Open the Power BI (.pbix) file and refresh the data.
7. Review the project report and presentation for detailed insights.
